# Install

{~ if wpf ~}
You can [get LiveCharts from Nuget](https://www.nuget.org/packages/LiveChartsCore.SkiaSharpView.WPF), using the package manager console:

> Install-Package LiveChartsCore.SkiaSharpView.WPF
{~ end ~}
